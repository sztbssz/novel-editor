# 🎭 网文桥段素材库

专业网文创作辅助工具，收集整理经典小说桥段，为创作者提供灵感参考。

| 🎭 身份类 | 7 | 扮猪吃老虎、身份揭露、重生穿越等 |
| ⚔️ 冲突类 | 9 | 装逼打脸、退婚流、复仇流等 |

| 💕 情感类 | 9 | 英雄救美、后宫修罗场、青梅竹马等 |
| 🧠 智斗类 | 7 | 运筹帷幄、借刀杀人、反间计等 |
| 💎 资源类 | 9 | 捡漏、夺宝、炼丹、炼器等 |
| 🤝 关系类 | 7 | 拜师学艺、结交兄弟、建立势力等 |
| 🌟 特殊类 | 8 | 气运之子、反派自救、快穿任务等 |

### 核心功能

- 🔍 **智能搜索** - 关键词搜索桥段标题/内容/书籍/标签
- 🏷️ **分类筛选** - 大分类+子类型二级筛选
- 🎲 **随机灵感** - 写作卡壳时一键获取随机桥段
- 📊 **使用统计** - 记录每个桥段被引用的次数
- ➕ **自定义添加** - 添加自己的桥段到数据库
- 📤 **数据导出** - 支持JSON格式导出备份

如需数据持久化和多设备同步：

```bash
cd trope-collector
npm install

```

然后在前端页面执行：
```javascript
localStorage.setItem('api_base_url', 'http://localhost:3000');
location.reload();
```

## 📖 使用指南

### 浏览桥段
1. 点击顶部分类标签筛选大类
2. 点击子类型标签进一步筛选
3. 使用搜索框快速查找

### 获取灵感
点击"🎲 随机灵感"按钮，系统会随机展示一个桥段。

### 添加桥段
1. 点击"+ 添加桥段"
2. 选择大分类和子类型
3. 填写标题和内容（1-2句话提炼）
4. 可选：填写来源书籍、作者、年份
5. 保存

### 使用桥段
点击桥段卡片上的 ✓ 按钮，使用次数+1。

点击 📋 按钮复制桥段内容。
